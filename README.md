# wordle-helper
A small node CLI to help solve [WORDLE puzzles](https://www.powerlanguage.co.uk/wordle/).

### To run:
```
cd ../to/this/repo
node index.js

>>> Placed letters? ---L-
>>> Known good? O
>>> Known bad? TISCEFRAUD

[
  'JOLLY', 'HOLLY',
  'LOWLY', 'WOOLY',
  'GOLLY', 'KNOLL',
  'NOBLY', 'LOLLY',
  'POLLY', 'JOWLY',
  'MOLLY'
]
```
